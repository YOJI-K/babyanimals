プライバシーポリシー

どうベビ（以下「本サイト」）は、ユーザーのプライバシーを尊重し、 個人情報の保護に努めます。本ポリシーでは、本サイトが収集する情報と その利用方法について説明します。

1. 収集する情報

本サイトは、以下の方法でユーザーの情報を収集することがあります。

  • アクセスログ：IPアドレス、ブラウザの種類、参照元URL、閲覧ページ、閲覧日時などの情報を自動的に収集します。
  • Cookie（クッキー）：アクセス解析のためにCookieを利用します。Cookieはユーザーの識別に使用される小さなテキストファイルですが、個人を特定する情報は含まれません。

2. Google アナリティクス（GA4）の利用

本サイトは、アクセス状況の把握・改善を目的として Google アナリティクス 4（GA4）を利用しています。 GA4 は Cookie を使用してユーザーの行動データを収集・分析しますが、 個人を特定する情報は収集しません。

収集するデータの例：

  • 閲覧したページのURL・タイトル
  • サイトの滞在時間・ページビュー数
  • 外部リンクのクリック（リンク先URL）
  • 赤ちゃん個別ページの閲覧（動物名・動物園名）
  • お気に入り登録操作
  • おおよその地域情報（都道府県レベル）

収集されたデータは Google のサーバーに送信・保存されます。 Google のデータ利用については Google プライバシーポリシー をご確認ください。

GA4 によるデータ収集を無効にしたい場合は、 Google アナリティクス オプトアウト アドオン をご利用いただくか、ブラウザの Cookie 設定で無効化できます。

3. Cloudflare Web Analytics

本サイトは Cloudflare Web Analytics も利用しています。 Cloudflare Web Analytics はプライバシーを重視した設計で、 Cookie を使用せず、個人を特定する情報も収集しません。 詳細は Cloudflare プライバシーポリシー をご参照ください。

4. Cookie の管理

ほとんどのブラウザでは、設定メニューから Cookie を管理・削除できます。 Cookie を無効にした場合、一部の機能が正常に動作しない場合があります。

5. 外部リンクについて

本サイトは動物園・ニュースメディア等の外部サイトへのリンクを含みます。 リンク先のプライバシーポリシーは各サイトの規定に従います。 本サイトは外部サイトの内容に関して責任を負いません。

6. 免責事項

本サイトは動物園・水族館等が公表した情報をもとに制作した非公式の情報サイトです。 掲載情報の正確性・最新性については保証しかねますので、 訪問前は各施設の公式情報をご確認ください。

7. プライバシーポリシーの変更

本ポリシーは必要に応じて改訂することがあります。 変更後のポリシーは本ページに掲載した時点で効力を持ちます。
