🐣 Baby Animals (Starter)

Cloudflare Pages + Workers + Supabase のスターターです。後日、Next.jsなどに置き換えてください。

API（Supabase経由の例）

news_items の最新10件を読み込むデモ（READはSupabaseのAnonKeyでOKですが、本スターターではデモのみに留めています）。

fetch('https://YOUR_SUPABASE_PROJECT.supabase.co/rest/v1/news_items?select=*&order=published_at.desc&limit=10', {
  headers: { apikey: 'YOUR_ANON_KEY' }
});
